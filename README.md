# 🤖 GenAI Persona App

An AI-powered chatbot built with **Next.js**, **TypeScript**, **Tailwind CSS**, and the **OpenAI API**. The application allows users to chat with AI personas that respond in a consistent personality and conversational style.

# 📖 About the Project

GenAI Persona App is a modern AI chatbot that demonstrates how to build a conversational interface using the OpenAI API and Next.js App Router.

The application focuses on:

- AI-powered conversations
- Persona-based responses
- Modern UI
- Clean architecture
- Server-side API integration

This project was built as part of a GenAI assignment to demonstrate prompt engineering, API integration, and modern frontend development practices.

# 🚀 Installation

## 1. Clone the Repository

```bash
git clone https://github.com/vikrantshishodia11-prog/GenAIPersonaApp.git
```

---

## 2. Navigate to the Project

```bash
cd GenAIPersonaApp
```

---

## 3. Install Dependencies

Using npm

```bash
npm install
```

---

## 4. Configure Environment Variables

Create a `.env` file in the project root.

Example:

```env
OPENAI_API_KEY=your_openai_api_key
```

---

## 5. Start the Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:3000
```

# 🧠 How It Works

1. User opens the application.
2. User enters a message.
3. The request is sent to the API route.
4. The API securely communicates with the OpenAI model.
5. The AI generates a response.
6. The response is displayed in the chat interface.
